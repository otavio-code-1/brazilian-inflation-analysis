# Brazilian Inflation Analysis (2000–2024)

Exploratory Data Analysis (EDA) of Brazil's official inflation index (IPCA) using R.

## Objectives

- Analyze monthly inflation
- Identify seasonal patterns
- Produce descriptive statistics
- Create professional visualizations

## Technologies

- R
- tidyverse
- ggplot2
- lubridate
- rbcb

## Data Source

Central Bank of Brazil (BCB)
Series 433 (IPCA)

## Results

The analysis suggests seasonal inflation behavior, with higher average inflation rates during the beginning and end of the year.
